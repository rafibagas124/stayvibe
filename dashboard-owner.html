📌 Cara mendapatkan link Maps yang benar: Cara 1 — Link Share biasa (paling mudah):
Buka Google Maps → cari lokasi → klik Bagikan → copy link-nya
Contoh: https://maps.app.goo.gl/xxxxx

Cara 2 — Link Embed (paling akurat):
Buka Google Maps → cari lokasi → klik Bagikan → Sematkan Peta → copy bagian src="..." dari iframe
Contoh: https://www.google.com/maps/embed?pb=!1m18...

Cara 3 — Koordinat saja:
Contoh: -6.917464, 107.619123

📖 Cara mendapatkan link video: YouTube: Buka video → Klik Share / Bagikan → Embed → copy bagian src="..." saja
Contoh: https://www.youtube.com/embed/KODE_VIDEO

Google Drive: Upload video → Klik kanan → Dapatkan link → ubah jadi Anyone with link → copy ID file-nya
Contoh: https://drive.google.com/file/d/ID_FILE/preview
